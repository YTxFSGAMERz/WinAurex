# Security Policy

## Supported Versions

We currently provide security updates and support for the `main` branch of the WinAurex OS Builder. 

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Forks   | :x:                |

## Reporting a Vulnerability

Security is a primary concern for the WinAurex project. Since this tool modifies operating system installation media, we take potential vulnerabilities extremely seriously.

If you discover a security vulnerability within the OS Builder scripts, please **DO NOT** disclose it publicly. 

Instead, contact the repository owner privately via:
* **Telegram:** [@YTxFSGAMERz](https://t.me/YTxFSGAMERz)
* **WhatsApp:** [Direct Contact](https://wa.me/917778906798)

Please provide a detailed description of the vulnerability, steps to reproduce it, and any potential impact. We will make every effort to respond rapidly and address the issue.
