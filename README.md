This repository contains a collection of tools, scripts, and configurations designed to optimize Windows 10/11 systems for better performance, privacy, and usability. It includes various utilities, registry tweaks, batch scripts, and an unattended Windows installation configuration.

---

## 🚀 Key Features

*   **Performance First:** Registry tweaks for CPU priority, SSD optimization, and custom power plans.
*   **Privacy Focused:** Comprehensive telemetry disabling and deep bloatware removal.
*   **Automated Setup:** Unattended installation configs that bypass TPM/RAM checks automatically.
*   **Curated Toolset:** Hand-picked utilities for hardware monitoring, drivers, and system management.

## 🛠️ Quick Usage

1.  **Basic Optimization:** Run `Tweaks\Apply Optimizations.bat` as administrator.
2.  **Drivers:** Use tools in `Drivers\` to update your hardware.
3.  **Privacy:** Apply tweaks in `Tweaks\Recursos\Telemetry\`.
4.  **Clean Install:** Use `autounattend.xml` with Windows Setup for a fresh, debloated system.
