# 🔑 Windows & Office Activators Guide

This directory contains highly reliable tools for activating various editions of the Microsoft Windows operating system and Microsoft Office suites.

---

## 🛠️ Available Activation Tools

### 1. MassGrave (Microsoft Activation Scripts - MAS)
*   **File**: `MassGrave.cmd`
*   **Source Project**: Open-source, widely trusted activation script repository.
*   **Core Activation Methods**:
    *   **HWID (Hardware ID)**: Obtains a genuine permanent digital license for Windows 10/11 Home, Pro, and Enterprise editions. This activation is bound to your system's motherboard and remains active even if you perform a clean reinstallation of the operating system.
    *   **KMS38**: Activates Windows 10/11/Server editions up to the year 2038. Highly recommended for Enterprise LTSC and Server configurations that do not support standard HWID licenses.
    *   **Online KMS**: Uses remote Key Management Service emulation to activate Windows and Microsoft Office products for 180 days. MAS creates an automated renewal task in Windows Task Scheduler to ensure the license updates automatically without user intervention.
*   **How to Use**:
    1. Right-click `MassGrave.cmd` and select **Run as administrator**.
    2. An interactive terminal menu will appear.
    3. Input `1` to apply permanent HWID activation, `2` for KMS38, or `3` to activate Microsoft Office via Online KMS.
    4. Wait for the green success message, then close the menu.

### 2. AAct x64
*   **File**: `AAct_x64.exe`
*   **Developer**: Ratiborus
*   **Description**: A compact, lightweight portable KMS activator designed specifically to active Volume License (VL) editions of Windows Vista, 7, 8, 8.1, 10, 11, and Office 2010 through 2021.
*   **How to Use**:
    1. Temporarily pause active antivirus real-time monitoring (as portable KMS key injectors are flagged as false-positives due to their active memory modifications).
    2. Right-click `AAct_x64.exe` and select **Run as administrator**.
    3. Click **Windows Activation** or **Office Activation** on the left menu.
    4. Wait for the terminal readout to display "Product activation successful."
    5. Re-enable your antivirus monitoring.
